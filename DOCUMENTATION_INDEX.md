## 📱 文档格式

大部分文档使用 Markdown 格式，可用任何文本编辑器或 Markdown 阅读器查看。

交互示例需要构建后在浏览器中打开：

```bash

open examples/advanced-features.html
```

所有源码包含完整的 JSDoc 注释，支持 IDE 智能提示。
